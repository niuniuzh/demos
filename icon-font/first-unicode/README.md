### unicode字体测试

### 使用方法

下载项目，运行`npm install`,`cd first-unicode`,然后运行`gulp Iconfont`。

添加以下代码在icon.css
```
.iconfont{
	font-family: "myfont";
		-webkit-font-smoothing: antialiased;
		-moz-osx-font-smoothing: grayscale;
	font-style: normal;
	font-variant: normal;
	font-weight: normal;
	text-decoration: none;
	text-transform: none;
	font-size: 40px;
    color: #f1006d;
}
```
